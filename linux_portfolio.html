Character Device and Linux

This project is centered on operating systems involves the enhancement of the system's functionality through two key components. Firstly, it encompasses the override of the original character device to a highly customizable one, allowing for versatile functionality tailored to specific needs. This customization empowers users to redefine the nature and capabilities of the character device, enabling it to assume diverse roles or functions beyond its default configuration. Additionally, the project involves modifying the Linux kernel to transition from its default real-time scheduling to a round-robin scheduling mechanism. This modification reshapes the kernel's task management strategy, prioritizing fairness and equal CPU time allocation among processes, ensuring a more balanced and equitable execution of tasks within the operating system. Overall, this collaborative endeavor aims to introduce flexibility and optimize task scheduling within the operating system, offering enhanced customization options and efficient resource utilization.
